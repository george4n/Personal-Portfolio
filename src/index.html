My Tech Stack

Navigating the dynamic world of web development, I’ve refined my toolset to combine both power and efficiency. Here's a snapshot of the technologies and tools I rely on to bring digital projects to life:

HTML5

The backbone of every web page I craft. With HTML, I lay down the structure, ensuring content is organized, accessible, and ready to be styled.

CSS3

My toolkit for design and aesthetics. With CSS, I bring the web to life, ensuring each page is responsive, visually engaging, and on-brand.

JavaScript

Adding interactivity and dynamism to every project. I ensure performance to create smooth user experiences.

SASS

My go-to CSS preprocessor. It allows me to write clean, reusable, and organized styles.

React

Modern JavaScript libraries/frameworks that empower me to build scalable and reactive user interfaces.

Git & Github

Version control is crucial. I use Git to track changes and collaborate seamlessly with fellow developers.

NPM

My go-to for managing project dependencies. With NPM, I streamline my development process, ensuring that every project utilizes the latest tools and libraries efficiently.

Netlify

My preferred deployment platform. With Netlify, I guarantee that my web projects are launched seamlessly, benefiting from continuous integration, automatic SSL, and global CDN distribution.
